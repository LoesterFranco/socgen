Description

This module divides a clock down by a programmed count and delivers a single enable out pulse as the counter reloads at 00. An enable input allows further rate scaling



Parameters

Name
 default

SIZE
 4
 Number of bits in down counter
RESET
 1
 value driven out during reset
SAMPLE
 0
 value in down counter for triggering output

 clock input

 count only occurs when enable is high
divider_in[]

 value of count minus one
divider_out
 output
 single clock wide pulse when count matches SAMPLE







Theory of Operation

The divider will load in the value from the divider_in port and count down one every clock where enable is high before reloading from the divider_in port. Dividers may be cascaded by connecting the output of one to the enable into of the next one.
