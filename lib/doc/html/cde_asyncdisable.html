Description

The cde_asyncdisable module will convert a active high synchronous reset pulse into an active low asychronous signal that is testable by scan testing. It inserted between the source of a generated reset and the active low reset_n port of a flipflop. A system wide active low asynchronous reset_n input provides a means to test the flipflops reset_n port and a optional system wide synchronous input is available

Theory of Operation

This module provides a means to break the synchronous reset path during scan testing by gating it with a atg signal. The local synchronous reset inputs MUST be glitchfree and should only come from the Q output of a flipflop
