Description

The cde_clk_gater provides a means to gate a clock from a synchronized enable signal. It is bypassed in scan mode using aatg signal

Theory of Operation

The cde_clk_gater will latch the enable signal during the clock low period so that the next rising edge can immediately progate to the output. This gating is bypassed when the atg_clk_mode signal is active. A scan observation flop may be added to this module to test the enable signal(s)
