Description

This module will take a signal that is asychronous to a clock domain and synchronize and delay it. The minimun allowed DEPTH is 2 stages.



Parameters

Name
 default
Description of what it does
WIDTH
 1
 sets the vector width of the data
DEPTH
 2
 sets the number of delay stages
RST_VAL
 1'b0
 sets the reset state for all stages

Theory of Operation

cde_sync_with_reset is a simple multistage metastable filter. Both data width and the number of stages are set via parameters. All stages may be reset to the same value using an asynchronous reset
