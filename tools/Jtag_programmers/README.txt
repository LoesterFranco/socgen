===================================================
Configuration steps for Jtag Programmers
===================================================

1) Digilent boards
 
   install urjtag and prog_usblaster script to reprogram it to usb_blaster
   requires 40-basic-permissions.rules file in /etc/udev/help.d
   run urjtag script in debug directory of synthesized part

2) Xilinx DLC_10

   requires setup file in /etc/udev/help.d
